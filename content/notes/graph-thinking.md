Graph thinking treats a note as part of a web instead of an isolated page.

Quartz turns links like [[public-notes]] into navigable edges. That makes the graph view and backlinks useful for following how an idea changes across essays, reading notes, and working papers.

The first version of this site keeps the graph intentionally small so the publishing path is easy to audit.
